# FLUTTER-APP - OneJourney

Mini version of OneJourney, an AI-powered Indian mobility super app built with Flutter.

## Includes

- Animated splash logo and four-slide onboarding
- Mock phone and OTP login with language selection
- Provider-powered journey mode, selected route, and bottom navigation state
- Home, Search, Wallet, Trips, and Profile tabs
- Direct and Combined route planning with filters
- Route result cards for Fastest, Cheapest, and Premium journeys
- Trip details, price breakdown, safety score, carbon points, payment, and QR ticket flow

## Setup

```bash
flutter pub get
flutter run
```

If this folder does not have platform folders on your machine, run this once after installing Flutter:

```bash
flutter create . --platforms=android,ios,web
```
